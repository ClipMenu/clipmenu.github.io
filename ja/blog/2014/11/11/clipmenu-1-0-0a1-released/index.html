ClipMenu 1.0.0a1 リリース

November 11, 2014

  • updates

本日、ClipMenuの新しいアルファ版をリリースしました。

※ ClipMenu 1.0.0a1はまだアルファテスト中です。この点に留意し、ご自身の責任においてご使用ください。

What's new?

  • 新生
  • パフォーマンスの向上
  • 記憶可能な履歴件数の増加
  • 履歴ウィンドウ
  • ポップオーバー・プレビュー
  • Yosemite ready

新生

新しいClipMenuはApple社の新言語Swiftによって一から書き直されました。これにより、今後長く安心してお使いいただける準備が整いました。

パフォーマンスの向上

ClipMenu 0.4.xでは履歴を常にメモリに保持していましたが、新しいClipMenuではデータをディスク上に保持し、必要に応じて取り出す方式に改められました。そのため、以前に比べて省メモリで動くようになりました。

記憶可能な履歴件数の増加

新しいClipMenuではCore Dataの採用により、より多くの件数の履歴を扱えるようになりました。

また、記憶する件数とメニューに表示する件数を別に設定できるようになりました。

履歴ウィンドウ

新しく追加された履歴ウィンドウから、履歴の一覧・検索・削除が一括して行えます。

履歴を選択してから'Touch'ボタンを押せば、最終更新日時が更新され、メニューの最上部に移動させることもできるようになりました。

ポップオーバー・プレビュー

新しいポップオーバー・プレビューはプレーンテキストだけでなく、リッチテキストや画像の表示もできるようになりました。プレビューが表示されるまでの待ち時間やアニメーションの可否は設定画面から行えます。

Yosemite ready

新しいClipMenuはYosemiteに完全対応しています。もちろん、ダークモードをオンにしても大丈夫です。

ダウンロード

ClipMenu-1.0.0a1.dmg

必要システム:

Mac OS X 10.9 (Mavericks) または Mac OS X 10.10 (Yosemite)

インストール:

  • 'ClipMenu.app' を /Applications か ~/Applications フォルダにコピーします。

アンインストール:

  1. ClipMenuを終了する。
  2. 下記のファイルとフォルダを削除:
    • ~/Library/Preferences/com.clipmenu.ClipMenu.plist
    • ~/Library/Application Support/com.clipmenu.ClipMenu
  3. ClipMenu.appを削除。
