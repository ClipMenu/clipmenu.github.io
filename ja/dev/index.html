ClipMenu 0.4.4a12 をダウンロード 別サイトからダウンロード

最終更新日: 2010-05-13 (JST)

変更点:

  • 0.4.4a12:

    • スニペット・エディタのツールバー・アイコンを変更（Suphi Aksoy氏作）。
    • スニペット・エディタのツールバー設定を記憶するように変更。

  • 0.4.4a11:

    • スニペット・エディタのレイアウトを変更。

  • 0.4.4a10:

    • ステータスバー・アイコンを追加（Suphi Aksoy氏作）。

  • 0.4.4a9:

    • Mac OS X 10.6.3の環境で、ClipMenuでペーストした後に、コマンドキーが押し下げられたままになる不具合を修正。

  • 0.4.4a8:

    • Dvorakキーボード配列を使用時に、Command-Vのキーストロークが正しく入力されない問題を修正。

  • 0.4.4a7:

    • 文字属性（下線）を変更するアクションの設定方法を、一部変更。
    • 0.4.4a6で導入して不具合のあった、環境設定画面を開く際にシステムの起動項目をチェックする動作を削除。

  • 0.4.4a6:

    • 起動項目への登録方法を、Apple EventからLaunch Servicesを使った方法に変更。
    • 環境設定画面を開く際に、起動項目にClipMenuが登録されているかどうかチェックして、チェックボックスに反映させるように変更。
    • 履歴メニューのショートカットキーが変更できなかった問題を修正。

  • 0.4.4a5:

    • 指定可能なRTFの文字属性を追加。
    • 文字属性の色をHTMLカラーネームで指定できるように変更。

  • 0.4.4a4:

    • RTFD形式を含んだ履歴の使用メモリを削減。

  • 0.4.4a3:

    • RTFの文字属性をアクションから変更するJavaScriptのメソッドを追加。

  • 0.4.4a2:

    • スニペットを管理するウインドウを環境設定から独立。

  • 0.4.4a1:

    • クリップボード履歴のみを表示する、「履歴メニュー」を追加（ショートカットキー: Command-Control-V）。
