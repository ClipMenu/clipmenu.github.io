ClipMenu

  • 機能 スクリーンショット 更新履歴 ヘルプ

スクリプト言語のJavaScriptによって、文字列操作を行います。

Case

  • Capitalize

    文字列の先頭だけを大文字にし、残りは小文字にする。

    適用前:

      my name is steve

    適用後:

      My Name Is Steve

  • lowercase

    文字列をすべて小文字にする。

    適用前:

      My name is Steve

    適用後:

      my name is steve

  • Title Case

    文字列をすべてタイトルケースにする。頭文字だけが大文字で、残りは小文字になるが、接続詞などは小文字になる。

    適用前:

      the chicago manual of style

    適用後:

      The Chicago Manual of Style

  • UPPERCASE

    文字列をすべて大文字にする。

    適用前:

      My name is Steve

    適用後:

      MY NAME IS STEVE

Trim

  • LTrim

    先頭のスペースを削除した文字列を返す。

    適用前:

      "  My name is Steve"

    適用後:

      "My name is Steve"

  • RTrim

    末尾のスペースを削除した文字列を返す。

    適用前:

      "My name is Steve  "

    適用後:

      "My name is Steve"

  • Trim

    前後のスペースを削除した文字列を返す。

    適用前:

      "  My name is Steve  "

    適用後:

      "My name is Steve"

暗号

  • MD5ハッシュを計算

    文字列から計算した、MD5ハッシュを返す。

    適用前:

      Apple

    適用後:

      9f6290f4436e5a2351f12e03b6433c3c

  • SHA-1ハッシュを計算

    文字列から計算した、SHA-1ハッシュを返す。

    適用前:

      Apple

    適用後:

      476432a3e85a0aa21c23f5abd2975a89b6820d63

  • Base64にエンコード

    文字列をBase64でエンコードする。

    適用前:

      Apple

    適用後:

      QXBwbGU=

  • Base64からデコード

    Base64でエンコードされた文字列をデコードする。

    適用前:

      QXBwbGU=

    適用後:

      Apple

HTML

  • 文字 → 数値文字参照

    文字を数値文字参照に変換する。

    適用前:

      Apple

    適用後:

      &#65;&#112;&#112;&#108;&#101;

  • 数値文字参照 → 文字

    数値文字参照を文字に変換する。

    適用前:

      &#65;&#112;&#112;&#108;&#101;

    適用後:

      Apple

  • Markdown → HTML

    Markdown形式の文字列を、HTML形式に変換する。

    適用前:

      A First Level Header
              ====================
            
              A Second Level Header
              ---------------------
            
              Now is the time for all good men to come to
              the aid of their country. This is just a
              regular paragraph.
            
              The quick brown fox jumped over the lazy
              dog's back.
            
              ### Header 3
            
              > This is a blockquote.
              >
              > This is the second paragraph in the blockquote.
              >
              > ## This is an H2 in a blockquote

    適用後:

      <h1>A First Level Header</h1>
            
              <h2>A Second Level Header</h2>
            
              <p>Now is the time for all good men to come to
              the aid of their country. This is just a
              regular paragraph.</p>
            
              <p>The quick brown fox jumped over the lazy
              dog's back.</p>
            
              <h3>Header 3</h3>
            
              <blockquote>
                <p>This is a blockquote.</p>
            
                <p>This is the second paragraph in the blockquote.</p>
            
                <h2>This is an H2 in a blockquote</h2>
              </blockquote>

  • URIコンポーネントをエンコード

    特定の文字の実体をエスケープシーケンスに置き換えて、Uniform Resource Identifier (URI)の要素をエンコードする。

    適用前:

      http://www.apple.com/

    適用後:

      http%3A%2F%2Fwww.apple.com%2F

  • URIコンポーネントをデコード

    Uniform Resource Identifier (URI)の要素をデコードする。

    適用前:

      http%3A%2F%2Fwww.apple.com%2F

    適用後:

      http://www.apple.com/

  • HTML文字実体参照にエスケープ

    < > & " ' を、HTML文字実体参照にエスケープする。

    適用前:

      <a href="http://www.apple.com/">Apple</a>

    適用後:

      &lt;a href=&quot;http://www.apple.com/&quot;&gt;Apple&lt;/a&gt;

  • HTML文字実体参照からアンエスケープ

    HTML文字実体参照をアンエスケープする。

    適用前:

      &lt;a href=&quot;http://www.apple.com/&quot;&gt;Apple&lt;/a&gt;

    適用後:

      <a href="http://www.apple.com/">Apple</a>

  • タグを除去

    HTMLタグを除去した文字列を返す。

    適用前:

      <a href="http://www.apple.com/">Apple</a>

    適用後:

      Apple

  • タグで囲む…

    対象文字列を、ダイアローグウィンドウに入力したHTMLタグで囲む。

    以下の例では、一行目はダイアローグに「p」と、二行目は「h1 class="top"」と入力している。

    適用前:

      Apple
              Apple

    適用後:

      <p>Apple</p>
              <h1 class="top">Apple</h1>
